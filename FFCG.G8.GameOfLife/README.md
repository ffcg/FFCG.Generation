För att bygga koden:

Skriv antingen 'npx webpack' eller bara 'webpack' om du har installerat webpack globalt. Då genereras index.html och web.js i mappen public. Kör appen genom att öppna public/index.html.

** UPPGIFT **

1. Ni ska kunna välja startposition
2. Ni ska kunna spara startpositioner till nästa körning. Det ska vara möjligt att spara på två olika sätt: 1. In memory 2. I browsern.
3. Det ska vara möjligt att ändra animeringshastighet. Detta ska också sparas.
4. När det tar slut ska man kunna köra på nytt. Slut innebär att det "fastnat", inte när det hamnat i en oändlig loop.
